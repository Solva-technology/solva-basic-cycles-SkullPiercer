# Упражнения на циклы `for` и `while`

Эти задания помогут вам научиться использовать циклы для решения различных задач, таких как обработка списков, подсчеты и работа с пользовательским вводом.

## Цель

Вам предстоит реализовать несколько функций в файле `main.py`, используя циклы `for` и `while`.

**Важно:** Как и в прошлый раз, обращайте внимание на комментарии в коде. Они подскажут, где брать данные и как возвращать результат для проверки тестами.

## Инструкция

### Шаг 1: Настройте окружение

```bash
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```

### Шаг 2: Выполните задания в `main.py`

Откройте `main.py` и реализуйте следующие функции:

1.  **`sum_list_elements(lst)`**: Используя цикл `for`, посчитайте и верните сумму всех чисел в списке.

2.  **`find_max_element(lst)`**: Найдите и верните максимальный элемент в списке. Не используйте встроенную функцию `max()`.

3.  **`count_even_numbers(lst)`**: Посчитайте и верните количество четных чисел в списке.

4.  **`create_string_from_list(lst)`**: Создайте и верните одну строку из всех элементов списка, соединенных вместе.

5.  **`sum_until_negative(lst)`**: Используя цикл `while`, считайте сумму чисел в списке до тех пор, пока не встретится отрицательное число.

### Шаг 3: Запустите тесты

После реализации функций запустите тесты, чтобы проверить себя:

```bash
pytest
```

Тесты помогут вам понять, где допущены ошибки, и убедиться, что все работает правильно.
